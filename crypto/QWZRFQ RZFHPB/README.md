# Writeup

简单的 Mono-Alphabetic Substitution Cipher。

- 方法一

  频率分析或熵分析。或直接猜测常出现单词再调整其余字母使得每个单词都能从字典上查到即可得到 substitution table。

- 方法二

  随便找个在线破译网站，原文和 flag 括号中内容一起丢进去分析即可直接解出 flag 内容。
