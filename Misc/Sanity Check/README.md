> 加入 IRC 群聊即可拿到本题 flag：
>
> 服务器 `irc.ctf.huangduligong.com`；
> 端口 `6667`；
> TLS 加密端口 `6697`。
> 若个别 IRC 客户端 TLS 端口出现证书无法信任的情况，请直接使用 `6667` 非加密端口。

## Writeup

直接任意 IRC 客户端连接服务器，再根据服务器的 MOTD (Message Of The Day) 的提示前往 `#CTF` 频道，频道的 Topic 即为本题 flag。
